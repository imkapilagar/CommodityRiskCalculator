## üåê Free Hosting Options

### **1. GitHub Pages (Best Option)**

‚úÖ **Pros:**
- 100% Free forever
- Fast & reliable
- Custom domain support
- Easy updates
- HTTPS included

**Steps:**

1. **Create GitHub Account** (if you don't have one)
   - Go to https://github.com
   - Sign up for free

2. **Create New Repository**
   - Click "New Repository"
   - Name: `commodity-calculator` (or any name)
   - Make it **Public**
   - Click "Create repository"

3. **Upload File**
   - Click "uploading an existing file"
   - Drag `commodity_risk.html` to upload
   - Commit changes

4. **Enable GitHub Pages**
   - Go to Settings ‚Üí Pages
   - Source: Deploy from branch `main`
   - Click Save

5. **Access Your Calculator**
   - URL will be: `https://YOUR-USERNAME.github.io/commodity-calculator/commodity_risk.html`
   - Bookmark this on your iPhone!
   - Works offline after first visit

**Update Anytime:**
- Just upload new version to GitHub
- Changes appear in 1-2 minutes

## üì≤ iPhone Access Methods

### **Method 1: Safari Bookmark**

1. Open your hosted URL in Safari
2. Tap Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Name it "Commodity Calculator"
5. Now it works like an app! üì±

### **Method 2: iCloud Drive**

If you want local access:

1. Save `commodity_risk.html` to iCloud Drive (from Mac)
2. Open Files app on iPhone
3. Navigate to file
4. Tap to open in Safari
5. **Note:** This works but requires internet first time

### **Method 3: Working Copy App**

For advanced users:

1. Install "Working Copy" app (free)
2. Clone your GitHub repo
3. Open HTML files directly
4. Works offline
